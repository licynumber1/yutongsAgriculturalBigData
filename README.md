# bdp

基于 create-react-app & antd-design-pro & mobx 实现的开箱即用的中台前端/设计解决方案。

因为 antd-design-pro 是一个很好的开箱即用的中台前端/设计解决方案，但是其基于 dva 构建，对 redux & redux-saga 用户友好，

我们决定使用 mobx ，而且舍弃掉其一些 CI/CD 的解决方案，所以决定基于 CRA 搭建。

# 支持

1. antd 按需加载
2. mobx 装饰器语法支持
3. less 支持
4. antd 默认主题覆盖
5. flow 支持

more ...

# 使用

```
git clone ...
yarn install 
yarn start
yarn run flow
yarn run build
```
